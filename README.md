## Tecnologias

* PHP - v8.2.28
* Docker - v28.0.4-1
* cURL - Integrado e nativo ao PHP v8
* Composer - v2.8.6

## Execução 

Em um ambiente docker execute a imagem do projeto, rode todos os serviços do composer. Abra na porta http://localhost:8080/index.php
